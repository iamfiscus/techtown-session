## Framework

**Phase 1 — Diverge: Generate ideas without judgment**
- Start from the seed ideas provided
- Expand to 20-50 total ideas by combining, extending, inverting, and riffing on the originals
- Include wild ideas — they often spark practical ones
- No evaluation yet. Quantity over quality. Every idea gets written down.

**Phase 2 — Cluster: Group into themes**
- Review all ideas and group them into 4-8 natural clusters
- Name each cluster with a clear, descriptive label
- An idea can only belong to one cluster
- If an idea doesn't fit anywhere, create an "Other / Wildcards" cluster

**Phase 3 — Converge: Score and select**
- Score each cluster on three criteria using a 1-10 scale:
  - **Impact** (1-10): How much could this reduce ticket volume or handle time?
  - **Feasibility** (1-10): How realistic is this to implement with a small team in 90 days?
  - **Speed-to-value** (1-10): How quickly would we see measurable results?
- Calculate a composite score for each cluster (simple average of the three scores)
- Rank all clusters by composite score
- Select the **top 3** and explain why they won
- For each winner, identify the single best concrete idea within the cluster to pursue first
