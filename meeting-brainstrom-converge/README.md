# Meeting --> Brainstorm --> Fermi --> Build

A 45-minute working session that turns a messy meeting transcript into a scored decision dashboard. Uses portable AI skills (brainstorm-diverge-converge + Fermi estimation) that run in both Claude Desktop and ChatGPT, then builds a visual dashboard in Hostinger Horizons. Attendees follow along and leave with their own artifact.

## Pre-Session Setup Checklist (60 seconds)

- [ ] **Claude Desktop** with thinking-frameworks skills installed ([guide](guides/install-claude-desktop.md)) OR **ChatGPT** ready ([guide](guides/use-in-chatgpt.md))
- [ ] **Hostinger Horizons** account -- free tier works ([horizons.hostinger.com](https://horizons.hostinger.com))
- [ ] This repo downloaded or open in your browser, with the VTT file accessible
