# Install Thinking-Frameworks Skills in Claude Desktop

1. Open **Claude Desktop** on your computer.
2. Click the **hamburger menu** (top-left) and go to **Settings**.
3. Navigate to the **Skills** or **Plugins** section (label varies by version).
4. Click **Add Skill** or **Browse Skills**.
5. Search for **"thinking-frameworks"** or enter the repo URL: `github.com/lyndonkl/claude`
6. Click **Install**. Wait for the confirmation.
7. Verify: start a new conversation and type **"brainstorm"**. You should see the brainstorm-diverge-converge skill activate with its phased framework.
8. You now have access to all thinking-frameworks skills: brainstorm, Fermi estimation, decision matrix, and more.

**Troubleshooting:** If the skill does not appear, restart Claude Desktop and try again. Check that you are on the latest version.

> Don't have Claude Desktop? See [use-in-chatgpt.md](use-in-chatgpt.md) for the browser alternative.
